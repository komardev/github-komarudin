### Clone repo

``` bash
# clone the repo
$ git clone https://github.com/komarudinn/github-komarudin.git

# go into app's directory
$ cd github-komarudin

# install app's dependencies
$ npm install

```

## Usage

``` bash
# serve with hot reload at localhost:3000
npm start

# build for production with minification
npm run build
